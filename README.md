# Library Management system(Python)
Project Description:

The Library Management System is a simple console-based Python project developed for 1st year students.
It helps manage books in a library using basic Object-Oriented Programming concepts.

The system allows the admin to:
Add books
View all books
Search books
Issue books
Return books

This project demonstrates the use of:
Classes and Objects
Loops
Conditional Statements
Functions
Menu-driven programming
